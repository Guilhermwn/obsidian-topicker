# obsidian-topicker

### Descrição
Conjunto de Jupyter Notebooks para criação de Mapas de Conteúdos(MOC) com base no título das notas no Vault.

### Como usar
Faça download do Notebook `obsidian_analysis.ipynb` e siga os passos descritos.
